Creating & Assigning Tags

Tags are Unity's way of labeling GameObjects for easy identification

1

Create a New Tag

First, we need to define a new tag called "Logic" in Unity's Tag Manager.

  • Select Logic Manager in the Hierarchy
  • In Inspector, find Tag dropdown (shows "Untagged")
  • Click dropdown â†’ Add Tag...
  • Click + button, type "Logic", and Save
2

Assign the Tag

Creating a tag doesn't automatically assign itâ€”you must do this manually!

  • Go back to Logic Manager in Hierarchy
  • Click Tag dropdown again
  • Select "Logic" from the list

Don't Forget This Step!

A common mistake is creating the tag but forgetting to assign it. The Logic Manager MUST have the "Logic" tag selected!

Declare the Reference

In PipeMiddleScript, create a public reference variable to hold the LogicScript connection.

using UnityEngine;

public class PipeMiddleScript : MonoBehaviour
{
    // Reference to connect to LogicScript
    public LogicScript logic;
}
2

Find and Connect in Start()

Use FindGameObjectWithTag to locate the Logic Manager, then GetComponent to access its script.

PipeMiddleScript.cs 
void Start()
{
    // Find the GameObject with "Logic" tag
    // Then get its LogicScript component
    logic = GameObject.FindGameObjectWithTag("Logic")
                      .GetComponent<LogicScript>();
}

Why This Works

This is equivalent to dragging a component in the Inspector, but happens automatically when the Prefab spawns!

3

Call addScore on Trigger

Now we can use the logic reference to call addScore when the bird passes through.

PipeMiddleScript.cs (Complete) 
using UnityEngine;

public class PipeMiddleScript : MonoBehaviour
{
    public LogicScript logic;

    void Start()
    {
        logic = GameObject.FindGameObjectWithTag("Logic")
                          .GetComponent<LogicScript>();
    }

    private void OnTriggerEnter2D(Collider2D collision)
    {
        logic.addScore();
    }
}

04 â€¢ LAYER FILTERING

Only Bird Triggers Score

Use Layers to ensure only the Bird can trigger the scoring zone

ðŸŽ¯ Why Layer Filtering?

Right now, ANY object entering the trigger zone will increase the score. We want ONLY the Bird to trigger scoringâ€”not other colliders that might accidentally enter.

1

Create Bird Layer

Layers are like tags but used for physics and rendering filtering.

  • Select the Bird GameObject in Hierarchy
  • In Inspector, find Layer dropdown (shows "Default")
  • Click Add Layer...
  • In slot 3, type "Bird"
  • Go back to Bird, set Layer to "Bird"
2

Filter Collision by Layer

Add a condition to check if the colliding object is on Layer 3 (Bird).

PipeMiddleScript.cs 
private void OnTriggerEnter2D(Collider2D collision)
{
    // Check if the object is on Layer 3 (Bird)
    if (collision.gameObject.layer == 3)
    {
        logic.addScore(1);
    }
}

Layer Number 3

Layers 0-2 are reserved by Unity (Default, TransparentFX, Ignore Raycast). User layers start at 3, so our "Bird" layer is at index 3.

3

Add Parameter to addScore

Make the scoring system flexible by accepting a parameter for how many points to add.

LogicScript.cs (Updated) 
public void addScore(int scoreToAdd)
{
    playerScore = playerScore + scoreToAdd;
    scoreText.text = playerScore.ToString();
}

Now when calling addScore, pass the points value:

PipeMiddleScript.cs 
// Add 1 point (or any number you want!)
logic.addScore(1);

// Or add 5 points for bonus zones
logic.addScore(5);
