1

Create Logic Manager Object

Right-click in Hierarchy ‚Üí Create Empty. Name it "Logic Manager". This will hold our game state.

2

Create and Attach Logic Script

Create a new C# script called "LogicScript" and attach it to the Logic Manager object.

LogicScript.cs 
using UnityEngine;
using UnityEngine.UI;  // Required for UI components!

public class LogicScript : MonoBehaviour
{
    // Public variables appear in Inspector
    public int playerScore;
    public Text scoreText;

    // Context menu for testing in Editor
    [ContextMenu("Increase Score")]
    public void addScore()
    {
        playerScore = playerScore + 1;
        scoreText.text = playerScore.ToString();
    }
}

Pro Tip: Context Menu

The [ContextMenu] attribute lets you test functions directly from Unity's Inspector by clicking the three dots (‚ãÆ) on the component!

3

Connect UI to Script

This is crucial! Drag the Text object from the Hierarchy to the "Score Text" field in the Logic Script component.

  • Select Logic Manager in Hierarchy
  • Find the "Score Text" field in Inspector (under Logic Script)
  • Drag the Text object from Hierarchy into this field
  • Test with Context Menu ‚Üí "Increase Score"

Trigger-Based Scoring

Use invisible colliders to detect when the bird passes through pipes

üéØ Collider vs Trigger: What's the Difference?

Feature Regular Collider Trigger Collider
Physical Collision Yes No
Objects Pass Through No Yes
Detects Contact OnCollisionEnter OnTriggerEnter
Use Case Walls, Ground Scoring Zones, Pickups
1

Open the Pipe Prefab

Double-click your Pipe prefab to enter Prefab editing mode. We'll add a scoring zone between the pipes.

2

Create the Middle Trigger

Right-click in Hierarchy ‚Üí Create Empty. Name it "Middle". Position it in the gap between pipes.

  • Add Component ‚Üí Box Collider 2D
  • Resize the collider to fit the gap
  • Check "Is Trigger" in the Inspector!

Critical Step!

The "Is Trigger" checkbox is essential! Without it, the bird will physically collide with this invisible box instead of passing through.

3

Create the Pipe Middle Script

This script detects when the bird enters the trigger zone and calls the addScore function.

PipeMiddleScript.cs 
using UnityEngine;

public class PipeMiddleScript : MonoBehaviour
{
    public LogicScript logic;

    void Start()
    {
        // Find the Logic Manager in the scene
        logic = GameObject.FindGameObjectWithTag("Logic")
                         .GetComponent<LogicScript>();
    }

    // Called when another collider enters this trigger
    private void OnTriggerEnter2D(Collider2D collision)
    {
        logic.addScore();
    }
}
4

Tag the Logic Manager

For FindGameObjectWithTag to work, we need to tag our Logic Manager.

  • Select Logic Manager in Hierarchy
  • Click "Tag" dropdown in Inspector ‚Üí Add Tag
  • Create new tag called "Logic"
  • Assign this tag to Logic Manager
