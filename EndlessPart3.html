üéØ How it works

Setting direction.y = jumpForce gives the player an upward velocity. On the next FixedUpdate(), controller.Move() moves the player upward.

‚ö† Problem

Without gravity, the player flies forever after jumping! We need something to pull them back down.

Step 3 ‚Äî Gravity

Pulling the Player Down

We continuously reduce direction.y each frame by adding gravity multiplied by Time.deltaTime. This creates a natural falling arc.

Gravity (in Update) 
// Apply gravity every frame
direction.y += gravity * Time.deltaTime;

Frame 1: Jump

üìâ

Frame 2+: Gravity

direction.y += -20 * 0.016 = -0.32 each frame. The upward velocity shrinks, then goes negative.

‚¨á

Peak ‚Üí Fall

When direction.y crosses 0, the player starts falling. Gravity keeps accelerating them down ‚Äî just like real physics.

‚ö† New Problem

The player can now jump... but they can spam the up key mid-air for infinite jumps! We need a ground check.

Step 4 ‚Äî Ground Detection

isGrounded: Only Jump on Ground

The CharacterController has a built-in isGrounded property that returns true when the controller is touching a surface below it.

Ground Check 
if (controller.isGrounded)
{
    if (Input.GetKeyDown(KeyCode.UpArrow))
    {
        Jump();
    }
}
‚úÖ What isGrounded Does

Checks if the bottom of the CharacterController's capsule collider is touching any surface. Returns true ‚Üí allow jump. Returns false ‚Üí block jump.

üéÆ Result

Now the player can only jump when standing on the ground. Pressing Up Arrow mid-air does nothing ‚Äî no more infinite flying!

Step 5 ‚Äî Final Fix

Gravity Only When Airborne

We only want gravity pulling the player down when they're not on the ground. When grounded, we reset direction.y to a small downward force (-1) to keep the controller in contact with the ground.

Complete Jump + Gravity Logic 
if (controller.isGrounded)
{
    if (Input.GetKeyDown(KeyCode.UpArrow))
    {
        Jump();
    }
    else
    {
        direction.y = -1;  // Small downward force to stay grounded
    }
}
else
{
    direction.y += gravity * Time.deltaTime;  // Falling
}
üü¢

Grounded + Up Arrow

Call Jump() ‚Üí set direction.y = jumpForce. Player launches upward.

üîµ

Grounded + No Input

Set direction.y = -1. This tiny downward push keeps isGrounded returning true reliably.

üü†

In the Air

direction.y += gravity * deltaTime. Gravity accumulates, accelerating the player downward until they land.

Deep Dive

Why direction.y = -1?

This is a subtle but important detail that many tutorials skip. Here's why we use -1 instead of 0 when the player is on the ground.

‚ùå Using direction.y = 0

If the player walks off a ledge or over a bump, isGrounded may flicker between true/false. With y = 0, the controller can "float" for a frame before gravity kicks in ‚Äî causing jittery grounding.

‚úÖ Using direction.y = -1

A constant small downward push ensures the CharacterController is always being pressed into the ground surface. This makes isGrounded return true consistently and prevents floating.

üí° Think of it like this

It's like gently pressing your feet into the floor. You're not falling ‚Äî you're just making sure your shoes stay in contact with the ground so you can push off when you need to jump.

Complete Script

The Full PlayerController

PlayerController.cs ‚Äî With Jump & Gravity 
using UnityEngine;

public class PlayerController : MonoBehaviour
{
    private CharacterController controller;
    private Vector3 direction;
    public float forwardSpeed;

    private int desiredLane = 1;   // 0:LEFT  1:MIDDLE  2:RIGHT
    public float laneDistance = 4;

    public float jumpForce;
    public float gravity = -20;

    void Start()
    {
        controller = GetComponent<CharacterController>();
    }

    void Update()
    {
        direction.z = forwardSpeed;

        // Lane switching
        if (Input.GetKeyDown(KeyCode.RightArrow))
        {
            desiredLane++;
            if (desiredLane == 3)
                desiredLane = 2;
        }

        if (Input.GetKeyDown(KeyCode.LeftArrow))
        {
            desiredLane--;
            if (desiredLane == -1)
                desiredLane = 0;
        }

        float targetX = (desiredLane - 1) * laneDistance;
        direction.x = (targetX - transform.position.x) * 10f;

        // Jump & gravity
        if (controller.isGrounded)
        {
            if (Input.GetKeyDown(KeyCode.UpArrow))
            {
                Jump();
            }
            else
            {
                direction.y = -1;
            }
        }
        else
        {
            direction.y += gravity * Time.deltaTime;
        }
    }

    private void Jump()
    {
        direction.y = jumpForce;
    }

    private void FixedUpdate()
    {
        controller.Move(direction * Time.deltaTime);
    }
}
