Step 1

Opening Unity Hub

Your project management center

Unity Hub Interface Your starting point for all Unity projects
1 How to Start
  1. Launch Unity Hub on your computer
  2. Look for the blue "New project" button at the top right
  3. Click on it to begin creating your project
Why Unity Hub?

Unity Hub helps you manage multiple Unity versions, organize projects, and keep everything in one convenient place. It's essential for professional game development!

Step 2

Choose Your Template

Select Universal 2D Core for 2D games

2 Template Selection
  1. On the left side, click "All Templates"
  2. Browse through the available options
  3. Select "Universal 2D Core"
  4. This template is optimized for 2D game development

Perfect for Beginners:

  • Pre-configured for 2D games
  • Includes essential 2D components
  • Lighter and faster than 3D templates
  • Great for mobile game development
Step 3

Name Your Project

Choose a descriptive name

3 Project Configuration
  1. Enter a meaningful project name (e.g., "BirdLab")
  2. Choose the save location on your computer
  3. Verify your Unity username is correct
  4. Click "Create project" at the bottom
Naming Best Practices

Good names: "BirdLab", "Space_Shooter_2D", "Platformer_Game"

Avoid: "Test", "MyProject", generic names with numbers

Build Time: Unity will take a few seconds to build your project. This is normal as it sets up all necessary files!

Interface

The Unity Interface

Understanding the 4 main panels

Unity Editor Layout Four panels working together for game development

The Four Panels:

  • Hierarchy (Left): Lists all GameObjects in your scene
  • Scene (Center): Visual workspace for designing
  • Inspector (Right): Shows object properties
  • Project (Bottom): All your game assets
Panel 1

Hierarchy Panel

Scene organization at a glance

Hierarchy Panel (Left Side) Shows all GameObjects in your current scene
Default GameObjects

Every new project includes:

  • Main Camera: What the player sees
  • Global Light 2D: Illuminates your 2D scene

Try It: Click on GameObjects in the simulation's hierarchy to select them and see their properties!

Panel 2

Scene Panel

Your visual design workspace

Scene Panel (Center) Where you arrange and design your game world

Scene Tools:

  • Q - Hand Tool: Navigate the view
  • W - Move Tool: Reposition objects
  • E - Rotate Tool: Rotate objects
  • R - Scale Tool: Resize objects
Panel 3

Inspector Panel

Control object properties

Inspector Panel (Right Side) Displays detailed properties of selected objects
What You Can Edit
  • Transform: Position, Rotation, Scale
  • Components: Add functionality
  • Properties: Colors, materials, physics
  • Scripts: Attach custom behavior
