Course Description

This course explores game development and programming. Students will study programming design of game engines, the application of a physics engine, collision detection, screen graphics, debugging, and publishing. Topics include utilization of game assets, camera lighting, terrain textures, character controls, security, artificial intelligence (e.g., path finding) and the application of a modern gaming engine. Student projects are a significant course component.

üéØ

Instructional Objectives

  • Introduce students to basic game design, brainstorming, researching games, meet-ups, developer groups, beta testing and indie developer contests (e.g., Ludum Dare)
  • Introduce students to games as a user experience featuring storyline, graphics and audio assets, action scripts, and supporting software mechanisms
  • Familiarize students with a 3rd party gaming engine (Unity3D), including the engine's handling of projects, objects views, inspector windows, debugging, tool bars, and viewpoints
  • Provide students with sample games as case studies, familiarizing them with a variety of storylines and actual implementations
  • Familiarize students with game animation, including 3D math, boundaries, collision detection, and character movement
  • Introduce students to performance considerations, run-time profiling, and game security issues
  • Introduce different software features and structures that support game programming

Performance Objectives

  • Discuss and describe basic game design, game research, developer groups, beta testing and indie contests
  • Explain games in terms of storyline, graphics and audio assets, action scripts and supporting software mechanisms

Course Schedule

Week 1
Introduction to Game Engines
Hard-coded game vs. gaming engine, examples of game engines and architectures, importance of prototyping, basic architecture and frameworks, basic audio/sound, graphics, physics engines, supportive hardware (GPU), community support and resources, software and engine editor/interface.
Lab Assignment
Introduction to lab software, consideration of various websites and software repositories useful for game development
Week 2
Getting Started with Unity
Explore Unity, Real-time creation, Essentials of 3D and 2D game objects, Game assets, basic scripting, publishing the game, Unity Playground (2D game), Getting started with Playground, Project description
Week 3
Game Design & Prototyping
Basic game story design prototyping, storyboard, game objects and assets, using graphics and audio assets to perform simple modifications of animations and game sequences, triggers for programming. Final project discussion: requirements, guidelines, presentations
Lab Assignment
Essential Unity Concepts, Creator Kit: Puzzle (3D game development), Creator Kit: RPG (2D development)
Week 4
Physics & Scripting
Rigid bodies, physics engines and the implementation of special effects, modeling the effects of gravity, Unity Scripting fundamentals
Week 5
Game Mechanics & Controls
Game mechanics implementation, HUD (Heads-Up Display) design, Camera control systems and techniques
Week 6
Project Demo & Final
Student project demonstrations and presentations, Final examination
