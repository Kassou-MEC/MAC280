Scene View

Your 3D/2D workspace for building game worlds

What is the Scene View?

The Scene View is your interactive canvas for building game environments. It provides a 3D (or 2D) visualization where you can select, position, rotate, and scale GameObjects.

Unlike the Game View which shows the player's perspective, the Scene View gives you complete freedom to navigate anywhere in your scene.

Key Capabilities

  • Pan, zoom, and orbit around your scene freely
  • Select and transform objects visually
  • View scene from any angle or distance
  • Toggle between 2D and 3D modes
  • Use gizmos for precise positioning

Hierarchy Panel

Organize and structure your scene objects

Understanding the Hierarchy

The Hierarchy window displays every GameObject in your current scene as a tree structure. Each item in the scene has an entry in the hierarchy.

The power of the Hierarchy lies in showing parent-child relationships. When you drag one object onto another, it becomes a child, inheriting its transformations.

Key Capabilities

  • View all GameObjects in the current scene
  • Create parent-child relationships by dragging
  • Rename objects for better organization
  • Create new GameObjects (Empty, 3D, 2D, UI)
  • Search and filter objects by name

Game View

Preview your game as players will experience it

The Player's Perspective

The Game View renders your game exactly as it will appear to players. It displays what your game camera(s) capture.

When you press Play, the Game View becomes interactive‚Äîyou can test gameplay and see how systems work in real-time.

Key Capabilities

  • Real-time preview of rendered game output
  • Test different screen resolutions
  • Play mode for interactive testing
  • View stats: FPS, batches, triangles
  • Maximize on Play option
üîß

Inspector Panel

Configure every property of your GameObjects

The Property Editor

The Inspector is your control center for editing everything about a selected object. It displays all components‚ÄîTransform, physics, rendering, and custom scripts.

You can add new components, remove existing ones, and adjust values in real-time.

Key Capabilities

  • Edit Transform: position, rotation, scale
  • Add and remove components
  • Configure component properties
  • Assign references via drag-and-drop
  • Edit multiple selected objects at once
üìÅ

Project Panel

Your asset library and file management hub

Managing Your Assets

The Project panel is your file browser for all assets in your Unity project. Unlike the Hierarchy, it displays every asset available‚Äîwhether used or not.

Drag assets directly into the Scene or Hierarchy to use them.

Key Capabilities

  • Browse all project assets in folders
  • Import assets by dragging files in
  • Create new scripts, materials, prefabs
  • Search assets with filters
  • Create and manage Prefabs
üíª

Console Panel

Debug output and error tracking

Your Debugging Window

The Console panel displays log messages, warnings, and errors from both Unity and your custom scripts. When something goes wrong, this is where you'll find clues.

Use Debug.Log() in your scripts to print custom messages.

Key Capabilities

  • View logs, warnings, and errors
  • Debug.Log() for custom messages
  • Click errors to jump to code
  • Filter by message type
  • Error Pause: stop play on errors
