Step 1

Unity Hub

Your project manager

Your Unity Hub Shows your projects: "My project (2)", AnimationTransition, InputSystem, and more

What Unity Hub Does

  • Manages all your Unity projects
  • Handles Unity Editor versions
  • One-click project creation
  • Template selection

Getting Started

  1. Download Unity Hub
  2. Install Unity Editor (LTS version)
  3. Click "New project"
  4. Choose 2D or 3D template
The Interface

Unity Editor

Your complete workspace

Your Unity Editor Interface All windows visible: Hierarchy, Scene, Game, Project, Console, and Inspector

The 5 Essential Windows:

1. Hierarchy - Lists all objects in your scene

2. Scene View - Your 3D editing workspace

3. Game View - Player's perspective

4. Project - All your assets and files

5. Inspector - Edit selected object properties

Window 1

Hierarchy Window

Scene organization

Your Hierarchy SampleScene with Main Camera and Global Light 2D

What's Here

  • SampleScene - Your current scene
  • Main Camera - What player sees
  • Global Light 2D - Lighting for 2D games

Organization Tips:

  • Use descriptive names
  • Group related objects
  • Create empty parents for organization
Window 2

Scene View

Your 3D workspace

Your Scene View Grid workspace with transform tools on the left, camera frame in center

The Transform Tools (Left)

  • Q - Hand: Pan view
  • W - Move: Move objects
  • E - Rotate: Rotate objects
  • R - Scale: Change size
  • T - Rect: 2D/UI positioning

Navigation

  • Right Mouse + WASD: Fly through scene
  • Middle Mouse: Pan
  • Scroll: Zoom
  • Alt + Left Mouse: Orbit
  • F: Frame selected
Tools

Transform Tools

Essential for object manipulation

Your Transform Toolbar Hand, Move, Rotate, Scale, Rect, and Transform tools

Keyboard Shortcuts

Q - Hand Tool (view only)

W - Move Tool (most used)

E - Rotate Tool

R - Scale Tool

T - Rect Tool (2D/UI)

Y - Transform (all-in-one)
